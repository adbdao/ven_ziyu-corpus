<節>般若義理的探究　　　　　　　　　　　　　[第四頁]</節>
  分別解說----一諦
　　具壽善現復分別請問於佛，何緣一切相智自性無？何緣色受想行識自性無？…
…佛分別回答如下：
　　<經>【佛言善現。一切相智無和合自性故。若法無和合自性。是法則以無性為性。
】</經>
　　諸法既是因緣和合，故無自性。若法已有自性，何待因緣和合？故一切法無
和
合自性。
　　<經>【善現。色受想行識無和合自性故。若法無和合自性。是法則以無性為性。】</經>
　　<經>【善現。眼處耳鼻舌身意處無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。色處聲香味觸法處無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。眼界耳鼻舌身意界無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。色界聲香味觸法界無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。眼識界耳鼻舌身意識界無和合自性故。若法無和合自性。是法則以
無
性為性。】</經>
　　<經>【善現。眼觸耳鼻舌身意觸無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。眼觸為緣所生諸受耳鼻舌身意觸為緣所生諸受無和合自性故。若法
無
和合自性。是法則以無性為性。】</經>
　　<經>【善現。地界水火風空識界無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。無明乃至老死愁歎苦憂惱無和合自性故。若法無和合自性。是法則
以
無性為性。】</經>
　　<經>【善現。內法外法無和合自性故。若法無和合自性。是法則以無性為性。】</經>
　　<經>【善現。四靜慮四無量四無色定無和合自性故。若法無和合自性。是法則以
無
性為性。】</經>
　　<經>【善現。四念住乃至八聖道支無和合自性故。若法無和合自性。是法則以無
性
為性。】</經>
　　<經>【善現。空解脫門無相無願解脫門無和合自性故。若法無和合自性。是法則
以
無性為性。】</經>
　　<經>【善現。八解脫乃至十遍處無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。布施波羅蜜多乃至般若波羅蜜多無和合自性故。若法無和合自性。
是
法則以無性為性。】</經>
　　<經>【善現。內空乃至無性自性空無和合自性故。若法無和合自性。是法則以無
性
為性。】</經>
　　<經>【善現。苦聖諦集滅道聖諦無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。一切三摩地門一切陀羅尼門無和合自性故。若法無和合自性。是法
則
以無性為性。】</經>
　　<經>【善現。佛十力乃至十八佛不共法無和合自性故。若法無和合自性。是法則
以
無性為性。】</經>
　　<經>【善現。大慈大悲大喜大捨無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。無忘失法恒住捨性無和合自性故。若法無和合自性。是法則以無性
為
性。】</經>
　　<經>【善現。一切智道相智無和合自性故。若法無和合自性。是法則以無性為性。
】</經>
　　<經>【善現。初眼第二第三第四第五眼無和合自性故。若法無和合自性。是法則
以
無性為性。】</經>
　　<經>【善現。初神通第二第三第四第五第六神通無和合自性故。若法無和合自性。
是法則以無性為性。】</經>
　　<經>【善現。有為界無為界無和合自性故。若法無和合自性。是法則以無性為性。
善現。由是因緣。諸菩薩摩訶薩應知一切法皆以無性為其自性。】</經>
　　　　　　往
